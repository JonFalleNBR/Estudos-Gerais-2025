Here We are going to cover about the Space Memory taken by the Algorithym 
The Most Import thing to think is 
 - To decrease and Improve our time complexity, we often will increase / allocate more complexity to the Space Memory 
    -> This is called Time-Space Trade-Off

 Everything that uses Space Memory is 
     - Variables
     - functions calls (not the Function itself, but everytime  that its called)
     - Data Structures (Lists, Arrays, Maps, etc)
     - allocations
